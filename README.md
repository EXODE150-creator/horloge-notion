# 🕐 Horloge Française Personnalisable

Widget horloge en HTML/CSS/JS, conçu pour être intégré dans **Notion** via iframe.  
Affiche la date et l'heure en français avec de nombreuses options de personnalisation.

---

## ✨ Fonctionnalités

- ⏰ Affichage en temps réel — heure, jour et date en français
- 🎨 Panneau de personnalisation intégré (masquable)
- 🔤 Choix de la police parmi 10 familles
- 🌈 Couleur du texte personnalisable
- 🟦 Couleur et transparence de la boîte d'horloge
- 📐 Taille indépendante pour l'heure, le jour et la date
- 📱 Responsive (mobile & desktop)
- 🪟 Fond transparent — intégration parfaite dans Notion

---

## 🚀 Utilisation

### 1. Hébergement

Héberger le fichier `index.html` sur un service statique :

- [Netlify Drop](https://app.netlify.com/drop) *(glisser-déposer)*
- [GitHub Pages](https://pages.github.com/)
- [Vercel](https://vercel.com/)

### 2. Intégration dans Notion

Dans une page Notion :

1. Taper `/embed`
2. Coller l'URL publique du fichier hébergé
3. Valider — l'horloge s'affiche directement dans la page

---

## 🎛️ Panneau de personnalisation

Cliquer sur l'icône **☰** en haut à gauche de l'horloge pour ouvrir le panneau.  
Cliquer à nouveau sur **✕** pour le masquer.

| Option | Description |
|---|---|
| Couleur du texte | Couleur de tous les éléments textuels |
| Couleur de la boîte | Teinte de fond de l'horloge |
| Transparence | Opacité du fond (0% à 100%) |
| Police | 10 familles disponibles |
| Taille de l'heure | De 30px à 120px |
| Taille du jour | De 16px à 60px |
| Taille de la date | De 12px à 40px |
